# AgentClinic Tech Stack

## Architecture
- **Language:** TypeScript (for reliability and developer experience).
- **Backend Framework:** Hono (Fast, lightweight, and type-safe middleware/routing).
- **Frontend Framework:** React (Standard for building interactive dashboards).
- **Database:** SQLite (File-based, zero-config, sufficient for MVP).
- **ORM:** Drizzle (Type-safe SQL mapping).
- **LLM Integration:** Anthropic Claude (via SDK) for Triage and Diagnosis reasoning.

## Deployment & Development
- **Package Manager:** npm.
- **Build Tool:** TypeScript Compiler (tsc).
- **Modern Browser Support:** Target latest versions of Chrome, Firefox, Safari, and Edge.
