# AgentClinic Roadmap

## Phase 1: Foundation (Current)
- Establish the "Constitution" (Mission, Tech Stack, Roadmap).
- Setup project structure and core dependencies.

## Phase 2: Patient Registration
- Create API endpoints for registering agents as patients.
- Implement basic SQLite persistence for patient records.

## Phase 3: Clinical Visit API
- Implement `POST /visits` endpoint.
- Basic Triage logic (simple severity scoring).
- Mock Diagnosis and Prescription responses.

## Phase 4: AI Diagnosis Engine
- Integrate LLM for natural language symptom analysis.
- Build the Ailment Catalog matching logic.

## Phase 5: Dashboard MVP
- Build a web interface to view patients and visit history.
- "Attractive" styling for Marketing approval.

## Phase 6: Follow-up & Effectiveness
- Implement follow-up reporting API.
- Track treatment success rates and update Ailment Catalog metadata.
